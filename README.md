# SentimentScope

A simple sentiment analysis tool for analyzing customer reviews.

## What it does

This app helps you analyze customer reviews to understand if they are positive, negative, or neutral. You can upload CSV files with reviews or enter text directly.

## How to use

1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Create a `.env` file with your database settings
4. Run `python init_db.py` to set up the database
5. Run `python app.py` to start the app
6. Open your browser and go to `http://localhost:5000`

## Features

- Upload CSV files with reviews
- Analyze text sentiment
- View results in a simple dashboard
- Download analysis results
